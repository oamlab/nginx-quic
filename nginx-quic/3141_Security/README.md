- ----------------------------

## 网络安全
### 网络安全威胁：

#### 1、主机内
- 对操作系统的威胁。

#### 2、接口处
- 对网站接口的威胁。

### 网络安全防御计划：
#### 1、防御预埋
- 建立防火墙，允许对端口80和端口443的访问，不允许对其非必要他端口的访问。

#### 2、自动防御
- 监控对操作系统的攻击和对网站接口的攻击，分析源IP，纳入黑名单。
- 运用字典库，拦截非法请求。
- 监控业务异常，自动预警上报，纳入分析处理清单。

#### 3、主动巡检
- 建立巡检清单，定时巡检。
- 攻击溯源。
